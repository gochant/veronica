方法

launch

启动一个应用程序

app.launch().done(function(){
    // 启动后执行
});

stop

停止一个应用程序

addExtension

  • addExtension(extension)

添加扩展，

app.addExtension(function(app){
    app.basic.say = function(){

    };
});

emit

广播消息

该方法与 app.sandbox.emit 的区别是：该方法会将应用程序名称作为最后一个参数附加到参数列表最后

app.emit('foo', 'bar');

// 部件监听代码：
sandbox.on('foo', function(param, appName){

});

消息

layoutSwitching

(layoutName, appName)

开始切换布局时发布

layoutSwitched

(layoutName, appName)

appStarting

(layoutName, appName)

应用启动时发布

appStarted

(appName)

应用启动完毕，加载完默认布局和默认页面后发布

pageLoading

(pageName, appName)

页面加载时发布

pageLoaded

(pageName, appName)

页面加载后发布
