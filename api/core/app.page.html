app.page.start app.page.start([isLoadDefaultPage])

  • isLoadDefaultPage：是否加载默认页，默认为 false

开启页面呈现，如果没有路由器，则需要加载一个默认页，如果存在路由器，则根据路由器识别当前应该首先加载哪个页面

app.page.start();

app.page.change app.page.change(pageName)

切换页面

app.page.change('about'); // 切换到关于页面

app.page.get app.page.get(pageName)

获取某个页面配置

app.page.get('about');

app.page.add app.page.add(pageConfig)

添加页面配置

app.page.add({
    'about': {
        name: '关于',
        layout: 'default',
        widgets: []
    }
});

app.page.isActive app.page.isActive(pageName)

判断某个页面是否是当前活动页面
